Enunciado

La siguiente base de datos contiene informaci√≥n acerca de incautaciones de bebidas alcoh√≥licas fraudulentas y de contrabando en la ciudad de Medell√≠n en un mes, que afectan los recursos para Salud y Educaci√≥n en el Departamento de Antioquia y de acuerdo a los resultados se toma decisiones para aumentar o disminuir los controles.

Licores


La base de datos contiene las variables - TL (Tipo de Licor) - PI (Precio de incautaci√≥n: se refiere al precio de venta en el establecimiento por unidad) - GAE (Grados de Alcohol en etiqueta) - GAQ (grados de alcohol en prueba qu√≠mica) - CE (Cantidades estandarizadas: n√∫mero de unidades estandarizadas a 750 ml).

Tenga en cuenta los siguientes aspectos:

  • El precio total de la incautaci√≥n se calcula como la cantidad estandarizada por el precio de incautaci√≥n.
  • El ipoconsumo, es el impuesto que deja de percibir el Estado para salud y educaci√≥n, el cual se calcula como: \(GAQ \times CE \times 400\) (pesos).
  • El licor incautado se clasifica como ‚ÄúFraudulento‚Äù si los GAE son distintos a los GAQ y como ‚ÄúContrabando‚Äù si son iguales.

Con los datos observados se tienen las densidades acumuladas emp√≠ricas, estas sirven para realizar un c√°lculo aproximado de la probabilidad, lo que se intenta resolver es:

\[P(25000 \leq X_i \leq 35000) = P(X_i \leq 35000) - P(X_i \leq 25000)\]

Donde \(X\) es el Precio de Incautaci√≥n e \(i\) representa cada Tipo de Licor.

# La funci√≥n ecdf sirve para calcular probabilidades emp√≠ricas
datos %>% 
  group_by(TL) %>% 
  summarise(P25 = ecdf(PI)(25000), # P(X <= 25000)
            P35 = ecdf(PI)(35000), # P(X <= 35000)
            Probabilidad = P35 - P25) # P(25000 <= X <= 35000)

De acuerdo con los comportamientos, se ve un poco de simetr√≠a y un comportamiento en forma de campana, si asumimos que estas muestras provienen de una distribuci√≥n normal, entonces con el promedio y desviaci√≥n est√°ndar se puede estimar los par√°metros de la distribuci√≥n para poder calcular las probabilidades.

\[X_i \sim N(\mu_i,\sigma^2_i)\]

datos %>% 
  group_by(TL) %>% 
  summarise(mu = mean(PI), # Promedio para estimar mu
            sigma = sd(PI), # Desviaci√≥n est√°ndar para estimar sigma
            # Probabilidades de una normal, seg√∫n mu y sigma
            P25 = pnorm(q = 25000,mean = mu,sd = sigma),
            P35 = pnorm(q = 35000,mean = mu,sd = sigma),
            Probabilidad = P35 - P25) # P(25000 <= X <= 35000)
